# Learning Playwright Batch

This repository contains Playwright learning material and exercises organized by chapter.

## Structure

- `chapter_01_Basics/` - Basic Playwright setup and initial scripts.
- `chapter_02_Java_Concepts/` - Java-related concepts and examples.

## How to use

1. Install dependencies for your project if needed.
2. Run Playwright scripts from the chapter folders.
3. Explore the examples to learn Playwright test automation concepts.

## Notes

- This repository is intended as a learning workspace for Playwright and related automation topics.
